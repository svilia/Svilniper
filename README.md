# Svilniper - Modern Reconnaissance Tool

Sn1per inspired, black & orange themed, full-featured Python OSINT / Recon tool.

## Features
- Subdomain Enumeration
- Shodan Integration
- GitHub OSINT
- WHOIS & DNS
- Nmap wrapper
- Email harvesting
- HTML + JSON reports
- Rich CLI UI

## Installation
```bash
pip install -e .
svilniper init
```
